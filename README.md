# Snake

A classic Snake game built with Python and Pygame.

## How to Run

```bash
pip install pygame
python snake_game.py
```

## How to Play

| Key | Action |
|---|---|
| Arrow keys | Move the snake |
| P | Pause / Resume |
| SPACE | Start / Restart |
| ESC | Quit |

- Eat red food to grow and increase your score
- Avoid hitting the walls or yourself
- Speed increases every 5 points
- High score is saved between sessions

## Features

- Circular smooth-rendered snake with directional eyes
- Pulsing food animation
- Animated body color gradient
- Start menu and game-over screen
- Pause overlay
- Persistent high score tracking

## Requirements

- Python 3.6+
- Pygame
